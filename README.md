# Cineby Desktop

A desktop port of the Cineby website, bringing your favorite movie and TV show streaming experience to a native desktop application.

## 📖 About

Cineby Desktop is a standalone desktop application that wraps the Cineby web experience into a native desktop environment. It provides a seamless, app-like experience with additional desktop-specific features while maintaining the full functionality of the Cineby platform.

## ✨ Key Features

- **Native Desktop Experience** - Full-screen viewing, system tray integration, and native notifications
- **Enhanced Media Playback** - Optimized video player with desktop-specific controls
- **Picture-in-Picture Mode** - Continue watching while browsing other content
- **Keyboard Shortcuts** - Full keyboard control for navigation and playback
- **System Integration** - Media keys support, volume control, and playback controls
- **Faster Performance** - Optimized loading times and smoother navigation
- **Offline Capabilities** - Cache content and continue browsing without internet
- **Distraction-Free Mode** - Immersive full-screen viewing with minimal UI
